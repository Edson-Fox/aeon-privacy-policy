🛡️ Política de Privacidade — Aeon

Última atualização: 9 de abril de 2026

Esta política se aplica ao aplicativo Aeon disponível na Google Play Store.

O Aeon foi criado com um compromisso claro:
👉 respeitar sua privacidade, sua autonomia e seus dados.

Este aplicativo funciona como um reprodutor de mídia local. Isso significa que seus arquivos permanecem no seu dispositivo e não são enviados para servidores externos.

📱 Dados que o Aeon acessa

O Aeon não coleta dados pessoais como nome, e-mail, localização ou identificadores únicos.

O aplicativo pode acessar apenas:

  • 🎵 Arquivos de áudio armazenados no dispositivo
  • 🕒 Posição de reprodução de faixas de áudio

Esses dados são usados exclusivamente para o funcionamento do app.

🔐 Como seus dados são usados

  • Todos os dados permanecem localmente no dispositivo
  • Nenhuma informação é enviada para a internet
  • Nenhum dado é vendido, compartilhado ou analisado externamente
  • O Aeon funciona offline

📂 Continuidade de reprodução

O Aeon pode armazenar localmente a posição de reprodução de algumas faixas de áudio.

Isso permite que você retome uma música de onde parou anteriormente.

  • Ficam armazenadas apenas no dispositivo
  • Não são compartilhadas ou transmitidas
  • São usadas exclusivamente para melhorar a experiência

🚫 O que o Aeon não faz

  • Não usa trackers
  • Não usa analytics de terceiros
  • Não coleta dados para publicidade
  • Não cria perfis de usuário
  • Não acessa contatos, câmera ou microfone

🔑 Permissões

O Aeon solicita apenas permissões necessárias para acessar arquivos de mídia no dispositivo, respeitando a versão do Android.

Essas permissões podem ser revogadas a qualquer momento nas configurações do sistema.

🧭 Monetização ética (futuro)

  • Será opcional
  • Não afetará a privacidade
  • Não incluirá anúncios invasivos
  • Não envolverá coleta de dados

O objetivo é manter o Aeon justo, sustentável e honesto.

🧑‍💻 Compromisso do desenvolvedor

O Aeon é desenvolvido por uma pessoa real, com responsabilidade real.

Se algo mudar nesta política, o texto será atualizado de forma clara dentro do aplicativo.
